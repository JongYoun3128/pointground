# POINTGROUD 기업 홈페이지

ksnpick.com을 모티브로 제작한 현대적인 기업 홈페이지입니다.

## 사업 분야

-   OTT 플랫폼
-   쇼핑몰
-   렌탈 사업
-   소상공인 지원
-   프렌차이즈 카페

## 파일 구조

```
기업홈페이지/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 기능
└── README.md       # 이 파일
```

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 또는 Live Server를 사용하여 실행

## 주요 기능

-   반응형 디자인 (모바일, 태블릿, 데스크톱)
-   부드러운 스크롤 애니메이션
-   숫자 카운터 애니메이션
-   인터랙티브 UI 요소
-   섹션별 페이드인 효과

## 섹션 구성

1. **히어로 섹션**: 메인 비주얼 및 회사 소개
2. **사업 분야**: 5개 주요 사업 영역 소개
3. **기술**: AI 및 클라우드 기술 설명
4. **통계**: 주요 성과 지표
5. **연혁**: 회사 발전 히스토리
6. **파트너**: 협력사 소개
7. **리더십**: 경영진 소개
8. **뉴스**: 최신 소식
9. **문의**: 연락처 정보
10. **채용**: 인재 채용 안내

## 커스터마이징

-   `styles.css`에서 색상 변수 수정 가능
-   `index.html`에서 텍스트 내용 변경
-   `script.js`에서 인터랙션 동작 조정

## 브라우저 지원

-   Chrome (최신)
-   Firefox (최신)
-   Safari (최신)
-   Edge (최신)

## 기술 스택

-   HTML5
-   CSS3 (Flexbox, Grid)
-   Vanilla JavaScript
-   Intersection Observer API
