# POINTGROUD 기업 홈페이지

ksnpick.com을 모티브로 제작한 현대적인 기업 홈페이지입니다.

## 사업 분야

-   OTT 플랫폼
-   쇼핑몰
-   렌탈 사업
-   소상공인 지원
-   프렌차이즈 카페

## 파일 구조

```
기업홈페이지/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 기능
└── README.md       # 이 파일
```

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 또는 Live Server를 사용하여 실행

## 주요 기능

-   반응형 디자인 (모바일, 태블릿, 데스크톱)
-   부드러운 스크롤 애니메이션
-   숫자 카운터 애니메이션
-   인터랙티브 UI 요소
-   섹션별 페이드인 효과

## 커스터마이징

-   `styles.css`에서 색상 변수 수정 가능
-   `index.html`에서 텍스트 내용 변경
-   `script.js`에서 인터랙션 동작 조정

## 브라우저 지원

-   Chrome (최신)
-   Firefox (최신)
-   Safari (최신)
-   Edge (최신)

## 기술 스택

-   HTML5
-   CSS3 (Flexbox, Grid)
-   Vanilla JavaScript
-   Intersection Observer API
